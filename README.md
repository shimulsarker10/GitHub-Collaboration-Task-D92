# GitHub-Collaboration-Task-D92

This repository was created as part of the Git and GitHub laboratory task for Batch D92, Department of Computer Science and Engineering, Dhaka International University (DIU).

## Objective

The objective of this lab is to gain practical experience with Git and GitHub collaboration by working in a team environment.

### Learning Outcomes

- Create and manage a GitHub repository
- Add collaborators to a project
- Clone repositories to local machines
- Create and manage Git branches
- Implement Java classes with constructors, getters, and setters
- Use Git commands (`git add`, `git commit`, `git push`, `git pull`)
- Merge branches into the `main` branch
- Practice collaborative software development using GitHub

## Lab Tasks

1. Create a repository on GitHub.
2. Add collaborators to the repository.
3. Clone the repository to local machines.
4. Create individual branches for each contributor.
5. Add Java files containing constructors, getters, and setters.
6. Commit and push changes to remote branches.
7. Pull the latest changes from the remote repository.
8. Merge all branches into the `main` branch.
